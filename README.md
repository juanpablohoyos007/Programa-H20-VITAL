Hola! Esta es mi aplicación de H2O Vital, un proyecto que hice para aprender a usar Flutter. La idea de la app es ayudar a las personas a recordar por qué es importante tomar agua y darles consejos útiles de forma sencilla.

🚀 ¿Qué tiene la aplicación?
La app está dividida en 4 pantallas principales para que sea fácil de navegar:

Inicio (Home): Bienvenida con una imagen llamativa.

Beneficios: Una lista de cosas buenas que hace el agua por nosotros.

Calculadora: Una sección para ver cuánta agua tomar según la actividad.

Tips: Consejos rápidos para el día a día.

🛠️ Widgets que aprendí a usar (Mis logros)
Para este trabajo, usé varios widgets especiales que nos pidieron en clase:

Stack: Lo usé en la pantalla de inicio para poner el texto "VIDA EN CADA GOTA" justo encima de la imagen del agua.

ListView: Lo apliqué en la pantalla de Beneficios para que la lista se pueda deslizar (hacer scroll) si hay mucha información.

SizedBox: Lo usé en casi todas las pantallas para dejar espacios en blanco y que los elementos no queden todos pegados.

Expanded: Fue muy útil en la Calculadora para que los cuadros de "Sedentario" y "Activo" se repartan el ancho de la pantalla de forma igual.

Container con Estilo: En la sección de Tips, usé un contenedor con bordes redondeados y una pequeña sombra para que parezca una tarjeta.

📂 ¿Cómo está organizado mi código?
Traté de ser ordenado con las carpetas:

Screens: Aquí están los archivos de cada pantalla (home.dart, beneficios.dart, etc.).

Widgets: Aquí puse los pedacitos de código que se repiten, como el título y el stack de la imagen.
